# 企业人事管理系统 java220

## 项目概述

企业人事管理系统是基于Springboot+Vue的前后端分离项目。系统采用MVC三层体系结构，通过整合Spring、SpringBoot、MyBatis、Vue和Maven等技术栈实现。该系统以MySQL数据库为基础，构建了B/S架构的Web应用，界面简洁，操作便捷。

## 技术栈

- 前端：Vue.js
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven
- 其他：JavaEE, MyBatis

## 功能模块

- 人事管理
- 员工信息管理
- 考勤管理
- 薪资管理
- 权限控制

## 系统角色

- 系统管理员
- 人事专员
- 员工

## 运行环境

- JDK 1.8 或更高版本
- MySQL 5.7 或更高版本
- Maven 3.x 或更高版本
- Node.js 12.x 或更高版本

## 部署步骤

1. 安装后端依赖环境：JDK, MySQL, Maven
2. 初始化数据库：导入SQL文件
3. 编译后端代码：执行Maven构建命令
4. 安装前端依赖：使用Node.js安装依赖包
5. 编译前端代码：执行前端构建命令
6. 部署应用：将编译后的前端代码部署到Web服务器

## 目录结构

```
- backend                后端代码
  - src
    - main
      - java
      - resources
- frontend              前端代码
  - src
    - assets
    - components
    - views
- pom.xml               Maven项目文件
- db                    数据库相关文件
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 基于Springboot的微服务架构，提供高可用性和弹性
- MyBatis作为持久层框架，支持定制化SQL、存储过程以及高级映射
- Maven构建系统，自动化依赖管理和项目构建
- 严格的角色权限控制，确保系统安全可靠
